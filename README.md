# TheHeroExpansion

A content expansion mod for **Slay the Spire 2**, adding 50+ new cards across all five characters, a bunch enchantments, 11 relics, and 3 potions

**Author:** BlackHero20

## Enchantments

| Enchantment | Effect                                                                                                                            |
|-------------|-----------------------------------------------------------------------------------------------------------------------------------|
| **Infected** | Enchant the cards adjacent to this one with Infected. Infected cards gain 5% Damage and Block for each Infected card in your deck. |
| **Golden Ratio** | Whenever you end the turn with unspent Energy, play this card from anywhere in your deck.                                         |
| **Dreamy** | Whenever this card is played, add a copy to your Discard Pile.                                                                    |
| **Advantageous** | The first time you play this card this combat, gain X Energy.                                                                     |
| **Royal** | When this card is played, gain 1 Star.                                                                                            |
| **Jumpscare** | When this card is played, kill all enemies with at least as much Doom as HP.                                                      |
| **Ravenous** | When this card is played, Evoke your leftmost Orb.                                                                                |
